Torque semi-parabolic characteristicsÂ¶

The semi-parabolic head and torque characteristics are valid on the domain \((q, \omega) \ge 0\). If mode_car_Cr=2, the couple of parameters rh_coef rule the torque characteristic.

The parameters rh_coef[1] and rh_coef[2] correspond to \(b_2\) and \(b_1\), i.e. the coefficients of \(x^2\) and \(x\) in the characteristic.
The following figures show their effect on the mass flow rate evolution in the cooling system.

import numpy as np
from os.path import join
import sys
sys.path.append("../script")
import fmpy_sim

directory = "../script/fmu"
fmu_filename = "PumpMotor.fmu"
path_fmu = join(directory, fmu_filename)

# Establish parameter bounds

n_simu = 5

dic_nominal = {
    "Pump.rh_coef[1]": -3.7751,
    "Pump.rh_coef[2]": 3.61
}

dic_range = {
    "Pump.rh_coef[1]": np.linspace(-4, -3, n_simu),
    "Pump.rh_coef[2]": np.linspace(3, 4, n_simu)  
}

# Define simulation parameters

list_output = ["Pipe.Q"]
final_time = 50

kwargs = fmpy_sim.get_recommended_dict()
kwargs.update({"stop_time": final_time, "output": list_output})

# Plot the simulation

from bokeh.plotting import figure, show, output_notebook
from bokeh.palettes import Category10
colormap = Category10[n_simu]
output_notebook()
