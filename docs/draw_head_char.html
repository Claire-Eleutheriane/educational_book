Head semi-parabolic characteristicÂ¶

The semi-parabolic head and torque characteristics are valid on the domain \((q, \omega) \ge 0\).
If mode_car_hn=2, the couple of parameters hn_coef rule the head characteristic.
hn_coef[1] and hn_coef[2] correspond to \(a_2\) and \(a_0\), i.e. the square and constant coefficient of the characteristic.
The following figures show their effect on the mass flow rate evolution in the cooling system.

import numpy as np
from os.path import join
import sys
sys.path.append("../script")
import fmpy_sim

directory = "../script/fmu"
fmu_filename = "PumpMotor.fmu"
path_fmu = join(directory, fmu_filename)

# Establish parameter bounds

n_simu = 5

dic_nominal = {
    "Pump.hn_coef[1]": -88.67,
    "Pump.hn_coef[2]": 43.15
}

dic_range = {
    "Pump.hn_coef[1]": np.linspace(-100, -60, n_simu),
    "Pump.hn_coef[2]": np.linspace(20, 60, n_simu)
}

# Define simulation parameters

list_output = ["Pipe.Q"]
final_time = 50

kwargs = fmpy_sim.get_recommended_dict()
kwargs.update({"stop_time": final_time, "output": list_output})

# Plot the simulation

from bokeh.plotting import figure, show, output_notebook
from bokeh.palettes import Category10
output_notebook()

Changing hn_coef[1] obviously influences the nominal mass flow rate reached by the pump.
Changing hn_coef[2] affects the nominal mass flow rate as well as the pump risetime and overshoot.
